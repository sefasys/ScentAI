# Roadmap

## Current Research Preview

- Grounded multilingual recommendation pipeline
- Canonical identity resolution
- Hard filters and answer validation
- Modal scale-to-zero deployment
- Cloudflare same-origin gateway
- Functional React chat client
- Frozen 120-case evaluation pack

## Next

- Finalize the visual system and mobile chat ergonomics.
- Add a blind holdout evaluation set and broader human review.
- Improve alias and abbreviation coverage without product-specific hacks.
- Move sessions to durable storage before horizontal API scaling.
- Add streaming or progressive status updates where the model backend permits it.
- Publish reproducible artifact checksums and schemas.

## Later

- Optional user accounts and cross-device collections
- Saved preference profiles with explicit consent
- Cost-aware autoscaling and per-user quotas
- Model and dataset cards after licensing review
- Adapter and retrieval ablations
