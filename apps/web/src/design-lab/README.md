# ScentAI Design Lab

This isolated preview explores the approved editorial archive, botanical herbarium, and restrained
surreal-motion direction. It does not replace or deploy the production frontend.

Run the existing frontend development server:

```bash
cd apps/web
npm run dev -- --host 127.0.0.1 --port 5173
```

Open the complete preview at:

```text
http://127.0.0.1:5173/?preview=design
```

Direct screen URLs:

- `?preview=design&screen=welcome`
- `?preview=design&screen=conversation`
- `?preview=design&screen=preparing`

The preview uses only the visual assets currently bundled under
`apps/web/public/design-lab/`. Reference material and discarded experiments are intentionally not
part of the public source tree.
